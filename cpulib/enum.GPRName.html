Expand description

An enumeration representing General Purpose Register (GPR) names.

This enum includes register names for various sizes: 64-bit (RAX, RBX, …), 32-bit (EAX, EBX, …), 16-bit (AX, BX, …), and 8-bit (AH, AL, …).
