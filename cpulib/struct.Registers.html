Struct cpulib::Registers

source · 
pub struct Registers { /* private fields */ }
Expand description

Represents a collection of registers within a simulated CPU architecture.

This struct includes SIMD registers, general-purpose registers (GPRs), flag registers, and instruction pointers, along with methods to manipulate these registers.
